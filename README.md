# 🚀 Pre-Gate In Survey App

A comprehensive **Flutter-based logistics survey application** designed to streamline the Pre-Gate In process for containers.  
This app helps surveyors manage container details, transporter information, and photos with real-time validation, API integration, and offline-ready capabilities.

---

## 📖 Description

The **Pre-Gate In Survey App** is built to simplify and digitize container surveys before gate-in.  
It ensures accuracy, reduces paperwork, and integrates with backend APIs to validate ISO codes, shipping lines, container numbers, and weights.

The app also supports uploading attachments, capturing survey photos, and saving complete survey records seamlessly.

---

## ✨ Features

- 📅 **Survey Date & Time Picker** – Select accurate date and time for surveys.
- 📦 **Container Details Management** – Capture ISO codes, type, size, category, make, and location.
- ⚖️ **Weight Validation** – Validate gross, tare, and payload weights in real-time.
- 📋 **Transporter Tab** – Input vehicle number, license details, and transporter name with strict validation.
- 🖼️ **Photo Attachments** – Upload and compress container photos before saving.
- 🔍 **Field Validation** – Real-time validation with caution dialogs for ISO codes, shipping lines, container numbers, and more.
- 🌐 **API Integration** – Connected with backend APIs for validation and saving records.
- 📲 **Offline-first Design (Planned)** – Smooth user experience even in low-network conditions.
